# Proyecto C

Dentro de este pequeño proyecto, encontraremos una calculadora de conversor de monedas en el lenguaje de programación C, que tenga las opciones de convertir entre quetzales, dólares estadounidenses y euros.

## Pre-requisitos 

1. Instalar C
2. Instalar el compilador de C (build-essential)
3. Instalar Visual Studio Code
4. Anteriormente, instalar Virtual Box y el archivo ISO de Ubuntu para configurar una maquina virtual

## Pasos para su ejecucion: 

1. Iniciar la máquina virtual con Ubuntu desde el software de virtualización (por ejemplo, VirtualBox).
2. Iniciar sesión en el sistema operativo Ubuntu dentro de la máquina virtual.
3. Compilar con GCC (ej. gcc hola.c -o hola)
4. Generar archivo ejecutable (ej. ./hola)
5. Ejecutar el programa y ver el resultado
